# <p align="center">Artline</p>
Se desarrollará una API para una red social enfocada al ámbito artístico. Esta API permitirá registrar una cuenta, crear una biografía y eliminar la cuenta. A su vez, existe un moderador que pueda consultar las biografías de los artistas, active, desactive o elimine las cuentas de artistas.
